# GamePi13 LCD Setup Guide

This repository provides a step-by-step guide to set up the **Waveshare GamePi13 1.3" LCD** on a fresh **RetroPie Buster 4.8.9** installation.  

The guide focuses on **display setup only**. Audio and button configuration can be added later.

---

## Repository Contents

- [`GamePi13_LCD_Setup.md`](GamePi13_LCD_Setup.md)  
  The main guide with full commands, configuration steps, and tips to get the LCD working reliably on multiple Raspberry Pi/GamePi13 setups.

---

## Features

- Tested on multiple Pi Zero 2W units
- Works with RetroPie Buster 4.8.9
- Includes commands for precompiled fbcp (most reliable method)
- Detailed framebuffer and HDMI configuration steps
- Ready for duplication on new setups

---

## Usage

1. Open the repository
2. Read through `GamePi13_LCD_Setup.md`
3. Follow the steps exactly on a **fresh RetroPie install** for reliable results
